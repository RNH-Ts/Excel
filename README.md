# Bike Sales Analysis Dashboard (Excel)
Simple Excel Dashboard, completed in ca. 1 hour

## Overview

This project analyzes customer demographic and lifestyle data to identify factors influencing bike purchase decisions. Using Excel, I transformed raw data into a structured dashboard designed to support business decision-making.

The objective was to explore patterns in customer segments and present insights in a clear, management-friendly format.

---

## Dataset

The dataset contains 1,026 customer records including:

- Marital Status
- Gender
- Income
- Number of Children
- Education
- Occupation
- Home Ownership
- Number of Cars
- Commute Distance
- Region
- Age
- Purchased Bike (Yes/No)

The key target variable is **Purchased Bike**.

---

## Tools & Techniques

- Data cleaning and formatting
- Pivot Tables for aggregation and segmentation
- KPI calculations (e.g., average income by purchase status)
- Dashboard creation with interactive filters
- Data visualization using charts and summaries

---

## Key Business Questions

- Which customer segments are most likely to purchase a bike?
- Does income level influence purchasing behavior?
- How does commute distance impact buying decisions?
- Are there regional or demographic differences in purchase rates?

---

## Key Insights

- Customers with mid-to-higher income levels show stronger purchase behavior.
- Shorter commute distances are associated with increased bike purchases.
- Certain age groups demonstrate higher buying rates.
